# Flags de execução

```cpp
Allowed options:
  -h [ --help ]                   (Optional) produce help message
  -f [ --fast ]                   (Optional) specify if the time must go 15x 
                                  faster.
  -d [ --develop ]                (Optional) turn on the develop mode. the time
                                  doesn't count.
  -g [ --qtd_of_goals ] arg (=10) (Optional) specify the qtd of goals to end 
                                  the game. 10 to 100
  -s [ --setup_path ] arg         (Optional) specify a .csv setup file 
                                  containing the initial robots position
```
